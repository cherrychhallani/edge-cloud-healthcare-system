# Edge-Cloud Healthcare Microservices System

## Overview
A distributed healthcare system using microservices with edge-cloud architecture.

## Services
- Appointment Service (Edge)
- Clinic Gateway (Edge)
- Patient Record Service (Cloud)
- Billing Service (Cloud)
- Prescription Service (Cloud)

## Tech Stack
- Python (Flask)
- Docker
- Docker Compose

## How to Run
```bash
docker compose up --build
